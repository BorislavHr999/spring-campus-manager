# Campus Management System
A backend REST API for university administration designed with **Java & Spring Boot**.

### 🚀 Key Features
* **Core Management:** CRUD operations for Students, Courses, Professors, Departments, and Clubs.
* **Database Relations:** Implements One-to-One, One-to-Many, and Many-to-Many relationships using JPA.
* **Advanced Search:** Dynamic filtering of students by multiple criteria (Name + City + Club) using JPA Specifications.

### 🛠 Tech & Usage
* **Stack:** Spring Boot 3, Spring Data JPA, H2 Database (In-Memory), Maven.
* **Run:** Execute `./mvnw spring-boot:run` in the terminal. The API starts at `localhost:8080`.
